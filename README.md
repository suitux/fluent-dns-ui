# Fluent DNS UI

UI to manage [Fluent DNS](https://github.com/suitux/fluent-dns)

## Installation

Download this repository.

```
git clone https://github.com/suitux/fluent-dns-ui
```

## Usage

1. Start with:

```
npm start
```
